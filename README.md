🇧🇷 Sistema de Seleção de Personagem com HUD (RPG Maker MV)
Um sistema de seleção de personagem limpo e pronto para uso no RPG Maker MV.
Este plugin permite que o jogador escolha seu personagem inicial através de uma interface interativa, com personagens animados e um HUD central exibindo informações essenciais.
✨ Funcionalidades
Cena de seleção de personagem
Navegação via teclado (esquerda/direita, confirmar, cancelar)
Destaque no personagem selecionado
Animação no personagem ativo
HUD central exibindo:
Retrato (face)
Nome
Classe
Nível
HP / MP
Leve e fácil de integrar
🛠️ Instalação
Baixe o arquivo do plugin
Coloque na pasta do seu projeto:js/plugins/
Ative o plugin no Gerenciador de Plugins do RPG Maker MV
🎮 Como Usar
Use o seguinte comando de plugin:
SelecaoPersonagem abrir
Isso abrirá a tela de seleção de personagem.
⚙️ Configuração
Edite os IDs dos personagens diretamente no código:
this._idsDosHerois = [1, 2, 3, 4];
Certifique-se de que esses IDs existem no banco de dados.
📌 Requisitos
RPG Maker MV
Assets padrão (faces e personagens)
Character selection scene with animated sprites and HUD (portrait, class, HP, MP, level) for RPG Maker MV.
🎮 Character Selection System with HUD (RPG Maker MV)
A clean and ready-to-use character selection scene for RPG Maker MV.
This plugin allows players to choose their starting character through an interactive interface, with animated sprites and a centered HUD displaying essential information.
✨ Features
Character selection scene
Keyboard navigation (left/right, confirm, cancel)
Highlighted selected character
Animated selected character
Centered HUD with:
Portrait (face image)
Name
Class
Level
HP / MP
Lightweight and easy to integrate
🛠️ Installation
Download the plugin file
Place it inside your project folder:js/plugins/
Enable the plugin in the RPG Maker MV Plugin Manager
🎮 Usage
Use the following Plugin Command:
SelecaoPersonagem abrir
This will open the character selection screen.
⚙️ Configuration
Edit the character IDs directly in the plugin:
this._idsDosHerois = [1, 2, 3, 4];
Make sure these IDs exist in your database.
📌 Requirements
RPG Maker MV
Standard character and face assets
